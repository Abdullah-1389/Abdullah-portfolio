
# 👨‍💻 Abdullah Attique — Portfolio Website

A modern, highly interactive, and responsive portfolio website built with **React**, **Vite**, and **Tailwind CSS**[cite: 1, 2, 3, 4]. It showcases my technical skills, software engineering projects, and digital marketing expertise[cite: 4].

## ✨ Features
* **Dynamic Interactivity:** Smooth scrolling, custom cursor/spotlight effects, and elements that reveal on scroll[cite: 4].
* **Responsive Layout:** Tailored to work seamlessly on all screen sizes, from mobile devices to large monitors[cite: 4].
* **Project Showcase:** Filterable project sections categorized into AI/ML, Web, Mobile, and Business/Marketing[cite: 4].
* **Clean Code Architecture:** Structured modularly with React Hooks (`useState`, `useEffect`, `useMemo`, `useRef`) for optimal performance[cite: 4].

---

## 🛠️ Built With
* **Frontend:** React, Vite, Tailwind CSS, Lucide React (for icons)[cite: 1, 2, 3, 4]
* **Styling:** Custom component-based CSS with clean variables[cite: 4]
* **Hosting:** Vercel / Netlify

## ⚡ Getting Started (Local Development)

If you want to run this project locally, follow these steps:

1. **Clone the repository:**
```bash
git clone [https://github.com/Abdullah-1389/Abdullah-portfolio.git](https://github.com/Abdullah-1389/Abdullah-portfolio.git)
cd Abdullah-portfolio

```


2. **Install dependencies:**
```bash
npm install

```


3. **Start the development server:**
```bash
npm run dev

```


4. **Build for production:**
```bash
npm run build

```
